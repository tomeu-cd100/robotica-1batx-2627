# SA5 · micro:bit i MicroPython: un altre paradigma

| | |
|---|---|
| **Trimestre** | 2n |
| **Durada** | 7 h (3-4 sessions) |
| **Maquinari** | micro:bit + Micro:shield (sensors externs) |
| **Llenguatge** | MicroPython (amb MakeCode com a pont) |

## Vincle competencial
- **Competències específiques:** CE-R1 (principal); CE-R3 (secundària).
- **Criteris d'avaluació:** CA1.2, CA3.1.
- **Competències clau:** STEM, CD, CPSAA.

## Sabers (Bloc D)
Sintaxi de Python (indentació, variables, funcions); E/S de micro:bit (matriu LED, botons, acceleròmetre, brúixola, temperatura, llum); pins externs amb Micro:shield; **comunicació per ràdio**; comparació C/C++ ↔ Python.

## Objectius d'aprenentatge
1. Escriure programes en **MicroPython** amb la sintaxi correcta (indentació!).
2. Usar els sensors integrats de la micro:bit i perifèrics via Micro:shield.
3. Implementar **comunicació per ràdio** entre dues plaques.
4. **Comparar** la mateixa solució en C/C++ (Arduino) i Python (micro:bit).

## Repte/pregunta inicial
> *"La mateixa idea, dos llenguatges: què canvia entre programar en C++ i en Python?"*

## Seqüència de sessions

| Sessió | Activitats |
|---|---|
| **1** | Editor Python micro:bit. Matriu LED, botons. "Name badge" i animacions. Indentació i variables. |
| **2** | Sensors integrats: acceleròmetre (detector de moviment/passes), temperatura, llum (nightlight). Funcions en Python. |
| **3** | **Ràdio**: missatgeria/comandament entre dues micro:bit. Mini-projecte: "dau digital" o "pedra-paper-tisora" per ràdio. |
| **4 (opc./+ampliació)** | **Taula comparativa C++ ↔ Python** d'un mateix programa (p. ex. comptador). Reflexió sobre paradigmes. |

## Producte
Aplicació amb micro:bit que usa sensors i/o ràdio (comptapassos, nightlight, joc per ràdio) + taula comparativa C++/Python al quadern.

## Avaluació
- Instruments: producte + quadern (inclou comparativa) + repte de codi Python.
- Rúbriques: **R1**, **R4** (comparativa/documentació).

## Atenció a la diversitat
- **Bastida:** **MakeCode** (blocs) com a pont abans del codi Python; simulador integrat.
- **+ Ampliació:** xarxa de 3+ plaques; ús de dades de sensor per a decisions; mini-IA (vegeu SA8).
