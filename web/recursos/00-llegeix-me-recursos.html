Blocs temàtics inclosos

  1. Institucional · Currículum i didàctica (Catalunya): XTEC (programació i robòtica, pensament computacional), Servei Educatiu del Segrià (230 miniprojectes), ROBOLOT, mSchools, exemples d'aula reals.
  2. Institucional · Espanya (INTEF): REA "Programamos con Arduino", Code INTEF (Python), Aula en Abierto (micro:bit MakeCode/Python), monogràfic ArduinoBlocks.
  3. Arduino · plataformes i cursos: ArduinoBlocks, Arduino Education / Student Kit / CTC 101, documentació oficial.
  4. Arduino · tutorials i blogs: Aprendiendo Arduino (C++ avançat), Luis Llamas, Todotecnología.
  5. Simuladors: Tinkercad Circuits, Wokwi (C++ i MicroPython, ESP32/IoT).
  6. micro:bit: lliçons oficials, First lessons with Python, Python Editor, MakeCode, Code & AI, Open Roberta Lab.
  7. Python / Informàtica de nivell alt: Raspberry Pi Foundation, Isaac Computer Science, CS50 (AP/x/Python), Google Python Class, freeCodeCamp, Hello Python.
  8. Europa: EU Code Week, Scientix, European School Education Platform, Escornabot.
  9. Competicions (aplicació real i exigent): WRO, RoboCup Junior, FIRST Tech Challenge, FIRST Global, ROBOLOT.

Suggeriment d'ús per a una matèria de 2 h anuals (nivell alt)

  • 1r trimestre — Fonaments: electrònica + Arduino (C/C++) amb Tinkercad/Wokwi com a simulador → recursos 16-26, 10, 3, 17.
  • 2n trimestre — Sistemes i sensors: micro:bit + Python i sistemes de control/IoT → recursos 27-34, 14, 26.
  • 3r trimestre — Projecte/repte: projecte de robòtica (trajectòries, control programat, IA) i opció de competició → recursos 32, 49-54, 9.

(Aquest és un esquema orientatiu; encaixa amb la Competència 5 i el bloc "Automatització" de Tecnologia i Enginyeria I — vegeu la carpeta Normativa.)
