Mapa del curs · de cada SA a un objecte real

Vista de conjunt per projectar: què aprens a cada SA i quins objectes pots dissenyar-hi. Detall i briefing a 00_Banc_objectes_disseny.md.

   ┌────────────────────────────────────────────────────────────────────────────────┐

   │                   │ entrada→procés→sortida          │ lluminosa · 🗼 far de maqueta │
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤
SA2│  🌈 s'expressa     │ múltiples sortides, PWM, color  │ 🪔 làmpada RGB · 🟢 llum      │
   │                   │                                 │ d'estat · 🚦 semàfor educatiu │
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤
SA3│  👁️ percep         │ llegir sensors (entrada)        │ 🌃 llum automàtica · 🅿️ sensor│
   │                   │                                 │ d'aparcament · 🔒 alarma calaix│
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤
SA4│  💪 es mou         │ servos i motors (moviment)      │ 🚧 barrera · 🍬 dispensador ·  │
   │                   │                                 │ 🦾 braç robòtic               │
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤

   │                   │                                 │ 🎲 joc de butxaca             │

   │                   │                                 │ termòstat · 🐣 incubadora     │
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤

   │                   │                                 │ netejador · 🏁 seguidor línia │

   │                   │                                 │ d'aforament · ✋ domòtica gestos│
   ├───────────────────┼─────────────────────────────────┼──────────────────────────────┤
SA9│  🎓 ho integra tot │ projecte complet (en autonomia) │ ✨ objecte lliure amb carcassa │
   └────────────────────────────────────────────────────────────────────────────────┘

El fil del curs

   ENCENDRE  →  EXPRESSAR  →  PERCEBRE  →  MOURE  →  CONTROLAR  →  DESPLAÇAR  →  CONNECTAR  →  INTEGRAR
     SA1          SA2          SA3        SA4         SA6           SA7           SA8          SA9
                                          (SA5: el mateix, en un altre llenguatge i format portàtil)

De fer parpellejar un LED a dissenyar un objecte connectat i intel·ligent amb la seva carcassa. Cada pas afegeix una capacitat nova al sistema… i una idea nova d'objecte per crear.
